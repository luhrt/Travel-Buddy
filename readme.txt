TravelBuddy é um site de compartilhamento de vídeos e fotos de viagem, 
desenvolvido com JavaScript e Node.js.
A versão inicial inclui a estrutura base do site, 
com a página inicial concluída e alguns módulos em desenvolvimento.

Funcionalidades planejadas:
    - Compartilhamento de vídeos e fotos de viagens.
    - Página inicial com design básico.
    - Sistema de login básico.

Pré-requisitos
    - Node.js instalado.
    - XAMPP (atualmente necessário para o banco de dados, mas estamos migrando para .db).
